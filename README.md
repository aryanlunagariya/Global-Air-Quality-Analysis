# Global Air Quality Analysis

## Overview
This project analyzes global air pollution data to understand pollution levels, trends, distributions, and relationships between different pollutants using data visualization techniques.

## Dataset
The dataset contains air quality measurements across multiple cities, including:
- AQI (Air Quality Index)
- PM2.5
- PM10
- NO₂
- SO₂
- CO
- O₃
- City
- Timestamp
- Air Quality Category

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations Included
- Line plot for AQI trends over time  
- Bar chart of average AQI for top 10 polluted cities  
- Histogram of AQI distribution  
- Boxplot to identify outliers in pollution data  
- Scatter plot showing relationships between pollutants  
- Heatmap showing correlation between pollutants  
- Countplot of air quality categories
